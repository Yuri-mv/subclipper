Azure Media Services Subclipper editor - Develop

The widget is built using React + Redux + Redux-Saga + Webpack.

Prerequisites

Download and install Node JS v6.10.3 or greater.

The recommend IDE is Visual Studio Code. The following extensions will help you to encounter linter errors in the code and debug it:

  • EditorConfig for VS Code
  • ESLint
  • Debugger for Chrome

Guidelines

The projects follow the Airbnb JavaScript Style Guide, refer to it for further information about coding styles.

Building

To build the project first clone this repository in your machine and run:

npm install

To develop using Hot Reload: npm run start. The project will be available in http://localhost:8080/

To build the widget in development mode run npm run build, building in production mode is as simple as running npm run build:prod. The bits will be available inside the dist folder.

Debugging

The project makes use of two Chrome extensions for debugging porpouses:

  • React Developer Tools: Lets you inspect the React component hierarchy, including component props and state
